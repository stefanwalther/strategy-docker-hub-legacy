# strategy-docker-hub

> sammler.io strategy to fetch statistics from Docker Hub.

---

## About

_strategy-docker-hub_ collects historical stats from Docker Hub per organization, user and repository.

- Repositories per user/organization
- Per repository
  - Stars per day
  - Pulls per day
